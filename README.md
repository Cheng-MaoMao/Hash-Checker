# Hash-Checker

[简体中文](https://github.com/Cheng-MaoMao/Sennpai/blob/main/README.md)|[English](https://github.com/Cheng-MaoMao/Hash-Checker/blob/main/README_en.md)

# **软件介绍**

支持从MD5到SHA-3的哈希校验的计算和对比

可以同时选择多个文件和文件夹

* 支持选择单/多个文件或文件夹
* 如果选择文件夹，会递归处理文件夹下的所有文件

# 使用的技术

图形界面使用Java Swing实现

使用SwingWorker进行异步计算
